Kitchen Sink

The Kitchen Sink contains examples and documentation of brand elements, typography styles, individual components, and general helpers.

  • "All the things" (this page) shows examples of styles and components. Jump to a section using the "on this page" navigation menu on the right. navigation menu in the top right.
  • For full component documentation, see the component page link in the "Kitchen Sink" navigation menu on the left. hamburger navigation menu in the top left.
  • For example component usage on a templated web page layout, see home page layout or standard page layout. (These pages use PHP-based components. A PHP proxy server needs to be enabled; see the README file for instructions.)
  • All Foundation components (except for Orbit) are included in the Global 3.1 build, but not all are documented here. For Foundation components/additional styles and implementation documentation not listed here, see the Foundation docs.

You can also use the Kitchen Sink for development and testing when modifying the Global 3.1 codebase. See the README file for development notes.

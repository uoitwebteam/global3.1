Patterns

Framing arrows background pattern

Use the class .brand-arrow-pattern and {{> main-palette}} to add a framing arrows pattern (white arrows at 10% opacity) to an element's background.

{{#unlesspage 'index'}}

SASS usage

For SASS usage, background and arrow size can be controlled via the brand-arrow-pattern() mixin.

				
					@include brand-arrow-pattern($bg: $darkblue, $size: 40px);
Parameter Value Description
$bg color variable Background color
$size size in pixels Repeat pattern size (the size of 1 large arrow and 1 small arrow)
{{/unlesspage}}

Repeated arrows background pattern

Use the class .repeated-brand-arrow-pattern and {{> main-palette}} to add a repeated arrows pattern to an element's background. The arrow colour is scaled based on the background colour, either a lighter or darker version of the background colour.

{{#unlesspage 'index'}}

SASS usage

For SASS usage, background and arrow size can be controlled via the repeated-brand-arrow-pattern() mixin.

				
					@include repeated-brand-arrow-pattern($size: 150px, $repeat: repeat, $position: right bottom);
Parameter Value Description
$bg color variable Background color
$size size in pixels Repeat pattern size (the size of one arrow group)
$repeat repeat property CSS repeat property value
$position position property CSS position property value
{{/unlesspage}}

Repeated arrows in a single row

Use the class .repeated-brand-arrow-row to add a row of overhanging repeated arrows to the bottom of an element. In some cases, the class .arrow-on-top (which controls z-indexes) may be necessary, depending on the child elements in the container (e.g. if there is only an image in the container).

Arrow colour can be specified with the class .{color}-arrow, using {{> main-palette}}. If a colour is not specified, the default arrow color is white.

The pattern and arrow color class can be combined with the standard .bgcolor class; different colors can be specified for the background and the arrows.

By default, the row of arrows overhangs the bottom of the element and is cut off. To show the full arrows, add the class .overflow to allow element overflow visibility.


The arrows can be animated when in view with the .animate-in-view utility class. The arrows slide in from the left edge.

Animated shield symbol

Add an animated shield symbol using the <SVG> syntax below. Shield component colours can be adjusted by using the component prefix and {{> main-palette}}:

.bg-[colour] (note that the shield's wrapping div will require the same background colour)
.shield-[colour]
.arc-[colour]
.arrow-[colour]

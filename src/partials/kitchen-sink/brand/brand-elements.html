Brand elements

Brand arrow divider

Use the class .brand-arrow-divider to add three small brand arrows above an element as a section divider.

Container with brand arrow divider above

"More" bouncing arrow

Use the class .more-bouncing-arrow to display a downwards facing arrow that bounces, indicating to scroll down. Use {{> basic-palette}} to change the default orange colour of the arrow.

Arrow clipping mask

An arrow clipping mask can be added to the left/right edge of an image or coloured div with the class .arrow-clip. This brand element works best on wide and short elements, such as banner-type images, testimonials, or short callouts.

Reverse arrow

The arrow direction can be reversed with the .arrow-clip-reverse class. This works best when paired with the default, right-facing arrow clipping style.


Diagonal

The element can be tilted with the .arrow-clip-diagonal class. This works best on full-page-width elements/images.


Combining with other brand elements

Add a brand arrow row pattern to two arrow-clipped elements for a high impact visual break. This works best on full-page-width images.

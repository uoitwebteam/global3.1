Responsive embed

Wrap embedded content like iframe, video, object, or embed in a container with the class .responsive-embed to maintain the correct aspect ratio regardless of screen size. The embedded content will resize to fill the width of its container, while maintaining its original aspect ratio.

Note: YouTube and Vimeo iFrames are automatically wrapped in a .responsive-embed container on page load; the container/class do not need to be manually added unless a specific aspect ratio is required (such as with video banners).

Resize this window to see the video below auto-resize along with it.


			<div class="responsive-embed">
				<iframe width="640" height="360" src="https://www.youtube.com/embed/HXz08CyFE7E?rel=0&showinfo=0" frameborder="0" allowfullscreen></iframe> 
			</div>

Disable responsive embeds

To disable automatic responsive embeds on YouTube or Vimeo videos, add the class .non-responsive-embed OR an attribute of data-responsive-embed="false" to the embedded content.

Note: The CMS editor removes iFrame classes; use the data-responsive-embed="false" attribute.

The video below is embedded at the specified width and height. Resize this window; the video will not auto-resize and you will see black bars above/below the video content area when the screen is smaller than the original width of the video.

Aspect ratios

Add ratio classes to change the aspect ratio of responsive embeds. The default ratio is 4:3. The .widescreen class will change the container's aspect ratio to 16:9. The .banner16x5 class will change the container's aspect ratio to 16:5, or the standard 1920x600 banner aspect ratio.

Note: If embedding a YouTube or Vimeo video that requires a different aspect ratio than the default 4:3, the non-responsive-embed class or attribute will also be required.

Available aspect ratios are specified in the $responsive-embed-ratios map in scss/settings/_foundation-settings.scss: 


			$responsive-embed-ratios: (
			default: 4 by 3,
			widescreen: 16 by 9,
			banner16x5: 16 by 5,
			banner1920x600: 16 by 5,
			banner1920x650: 14.75 by 5,
			thumbnail3x2: 3 by 2
		);

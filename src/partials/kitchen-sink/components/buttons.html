SASS usage

For a hollow button on a coloured background, use the mixin hollow-button-on-color() (e.g. white hollow button on orange background)

			
				<button class="my-button">Button</button>
			
		
			 
				.my-button { 
					/// button-hollow-style() Foundation mixin 
					/// @param $color - color (default: $primary-color)
					/// @param $hover-lightness - percentage lightness of $color to be applied to border color on hover (default: $button-hollow-hover-lightness)
					@include button-hollow-style($white, 0);
					
					/// hollow-button-on-color() mixin
					/// @param $background-hover - background hover color; 80% transparency of specified color
					@include hollow-button-on-color($background-hover: $white) 
				}

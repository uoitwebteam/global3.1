Image with caption

Use <figure> and <figcaption> elements to style an image with a caption.


Alternatively, use the .caption class on an <img> element to auto-format an image with caption (auto-generates <figure> and <figcaption> elements). The caption pulls from the "alt" attribute text.


Caption can be combined with the thumbnail style.

Thumbnail style

Circle style

Use the .circle class to add a round crop to a square image. Can be combined with the thumbnail style or .border utility class.


The .circle class can also be applied to a rectangular image. However, if the .thumbnail or .border styles are also used, the image must be wrapped in a .has-circle-[thumbnail/border] container.

Also note the centered alignment and default block-level div; the block-level wrapping div can be modified with .inline-block.

Arrow clipping mask

An arrow clipping mask can be added to the left/right edge of an image or coloured div with the class .arrow-clip. This brand element works best on wide and short images.

The arrow direction can be reversed with the .arrow-clip-reverse class. This works best when paired with the default, right-facing arrow clipping style.

See brand elements for further style variations and documentation.

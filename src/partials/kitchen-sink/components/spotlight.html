Spotlight (feature module)

The spotlight feature module is a wrapper for any free-form HTML content, but also includes templated styles for an image with caption, video with caption, and a feature (background) image with a content box overlay.

Note: The examples below are shown with several different content elements for demonstration purposes only. Do not include this many elements in a spotlight module in actual use.

Spotlight with image and caption

Add the class .image to a spotlight module and put the <img> in a <figure> element and caption in a <figcaption> element.

Figcaption text in a paragraph or span tag is centered by default. To override this, add the class .align-[left/right] to the <figcaption> element. Other elements such as lists and blockquotes have standard left-alignment.

The size of the <img> can be set using standard width and height attributes. On small screens, the image is full width (minus spotlight padding).

Spotlight with video and caption

Add the class .video to a spotlight module and put the video embed code in a <figure> element and caption in a <figcaption> element.

Wrap the video embed code in a <div> for automatic responsive-embed video sizing. On small screens, the video is full width (minus spotlight padding).

Figcaption text in a paragraph or span tag is centered by default. To override this, add the class .align-[left/right] to the <figcaption> element. Other elements such as lists and blockquotes have standard left-alignment.

Feature image spotlight

The feature image spotlight style has a full-width background image with a content box floated on either the left or right side of the image.

Float box alignment can be specified using the class .align-[left/right] on the feature-float-box element.

Float box sizing can be specified using Foundation column sizing class names, such as small-5 large-6 on the feature-float-box element. On small screens, the float box is full width (minus surrounding margins).

Float box colour can be specified using {{> basic-palette}}.

The background image position can be specified with an inline CSS property, background-position, on the .feature-image element. If not specified, the default is center center.

Note that the HTML structure of the feature module spotlight is slightly different than the basic spotlight HTML structure.

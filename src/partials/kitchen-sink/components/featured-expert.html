Featured expert

Display a featured expert profile from the Expert Centre.

A specific expert can be displayed using the profile's CMS asset ID in the data-id attribute, or they can be randomized using data-id="random"

Profile elements to show can be specified in the data-display attribute. If the specified element doesn't exist for a particular profile, the element won't be displayed.

Include a module heading with the data-heading attribute.

Featured expert

Display a featured expert using the spotlight component as a general template.

Featured expert with feature image

Display a featured expert using the feature image spotlight component as a general template. Add featureimage as a data-display attribute option.

The featureimage display option can be included by default. If a profile image has a feature (banner) image, it will be displayed as shown below. If the profile doesn't have a feature image, the fallback will be the specified solid coloured background.

Inline featured expert

Display a featured expert inline with body text by specifying a width in the data-width attribute and adding a .float-[left/right] utility class.

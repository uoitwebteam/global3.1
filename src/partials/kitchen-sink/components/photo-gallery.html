Photo gallery

Photo galleries are built using the lightGallery v2 and Justified Gallery plugins.

Vertical and horizontal images can be combined and will be autosized to the default/specified height, with all images justified to fill the width of the container.

Clicking on an image in the gallery will open it in a lightbox. Image captions in the lightbox can be included by adding a data-sub-html attribute to the anchor element. HTML content can be placed inside.

{{#unlesspage 'index'}}

Basic HTML markup:


			<div class="gallery-grid">
				<a class="gallery-item" data-src="img/img1.jpg" data-sub-html="Caption">
						<img alt="img1" src="img/thumb1.jpg" />
				</a>
				<a class="gallery-item" data-src="img/img2.jpg" data-sub-html="Caption">
						<img alt="img2" src="img/thumb2.jpg" />
				</a>
				...
			</div>

{{/unlesspage}}

Override the default row height (180px) and margin spacing (5px) with data-height and data-margin attributes.

Inline gallery

Photos can be displayed in an inline gallery, with one image featured at a time in a slider. The inline gallery also runs on the lightGallery v2 plugin.

Photo galleries using legacy markup

The lightGallery v2 and Justified Gallery plugins are backwards compatible with legacy gallery (Masonry, lightGallery v1, and Swiper) HTML structure and CSS classes; legacy HTML and class names can be used (e.g. existing v3.0 CMS templates), but the gallery will be rendered using the updated plugins.

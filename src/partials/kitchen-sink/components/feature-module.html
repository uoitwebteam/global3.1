Feature module

Feature modules are a set of templated page components, which include:

  • news feed
  • events feed
  • quick links bar
  • social media links
  • spotlight

Modules are typically full width (expanded page layout), but can also be set to the page content width. There is extra padding around the content.

Component-specific classes are added to the <section> wrapper element.

CMS feature module block component

In addition to the templated modules listed above, any free-form xHTML CMS block can be placed in a feature module block component. The xHTML block is wrapped in the templated feature-module wrapper (inherting templated styles like extra padding, background colours and heading styles).

Breakpoint-based alignment

Add breakpoint size classes to the front of the text alignment class to only apply the style on the specified size screen and larger. For example, <p class="medium-text-center">...</p> will be left-aligned until medium screen sizes and up, at which point it will switch to center-aligned.

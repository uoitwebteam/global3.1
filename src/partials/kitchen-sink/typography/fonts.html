Fonts

Primary font: Ubuntu

In web usage, this font is primarily reserved for headings or feature copy. Ubuntu font files are imported from Google Fonts.

Ubuntu

  • https://fonts.googleapis.com/css?family=Ubuntu:400,700
  • CSS font-family: 'Ubuntu'
  • CSS font-weight: 400 or 700
  • SASS variable: $heading-font

Secondary font: ITC Franklin Gothic STD

In web usage, Franklin Gothic is used for the main body font. Franklin Gothic variations are included either from the atlas-h shared.ontariotechu.ca server or the ituomweb ontariotechu.ca server, depending on where the CSS build file will be used.

Note: Franklin Gothic Condensed is included, but this is mostly a legacy font face and should not be used in regular web content.

Franklin Gothic Book

  • https://[shared.]ontariotechu.ca/global/files/fonts/franklingothicbook/FranklinGothic-Book-webfont.[eot/svg/ttf/woff/woff2]
  • CSS font-family: 'Franklin Gothic'
  • CSS font-weight: 400 or normal
  • SASS variable: $body-font

Fallback font: Arial

Arial

In cases where Ubuntu or Franklin Gothic cannot be used (such as on third-party websites), Arial can be used instead.


Icon font: UOIT Icons

Web icons are rendered using a custom font family, UOIT Icons. See also: Icons.

  • https://[shared.]ontariotechu.ca/global/files/fonts/uoiticons/uoiticons.[eot/svg/ttf/woff]
  • CSS font-family: 'UOIT Icons'
  • CSS font-weight: 400 or normal
  • SASS variable: $icon-font
